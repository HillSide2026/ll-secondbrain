# Skill: Conversion Path Design

## Purpose
Define the steps through which prospects move from awareness to engagement.

## Inputs
- Offer
- Audience definition
- Available assets

## Process
1. Define funnel stages.
2. Map assets to stages.
3. Define conversion triggers.

## Outputs
- Conversion path map

## Constraints
- Must align with real buying behavior.

## Invocation
Used when designing marketing funnels.
