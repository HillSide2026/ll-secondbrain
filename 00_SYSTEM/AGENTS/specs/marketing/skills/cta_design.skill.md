# Skill: CTA Design

## Purpose
Design calls-to-action that move prospects toward engagement.

## Inputs
- Offer definition
- Audience stage
- Funnel context

## Process
1. Identify desired conversion action.
2. Align CTA with audience readiness.
3. Design clear action language.

## Outputs
- CTA statement
- CTA placement guidance

## Constraints
- CTA must reflect actual offer.

## Invocation
Used when constructing funnel conversion paths.
