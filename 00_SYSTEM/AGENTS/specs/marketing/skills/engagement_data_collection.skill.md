# Skill: Engagement Data Collection

## Purpose
Collect engagement signals from marketing channels and campaigns.

## Inputs
- Campaign identifier
- Channel list
- Performance metrics sources

## Process
1. Gather engagement metrics.
2. Collect audience responses.
3. Compile performance indicators.

## Outputs
- Engagement dataset
- Campaign performance snapshot

## Constraints
- Data collection only; no interpretation.

## Invocation
Used by Market Signal Agent during signal intake.
