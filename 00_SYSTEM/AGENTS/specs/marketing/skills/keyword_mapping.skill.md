# Skill: Keyword Mapping

## Purpose
Identify and organize search keywords relevant to target audiences and offers.

## Inputs
- Target audience
- Core topics
- Market terminology
- Existing content

## Process
1. Generate relevant keyword candidates.
2. Classify keywords by intent:
   - informational
   - navigational
   - transactional
3. Group keywords into topic clusters.
4. Map clusters to potential content assets.

## Outputs
- Keyword map
- Topic cluster structure

## Constraints
- Must align with actual market search behavior.
- Cannot invent misleading search phrases.

## Invocation
Used by SEO/Discoverability Agent when planning discoverability strategy.
