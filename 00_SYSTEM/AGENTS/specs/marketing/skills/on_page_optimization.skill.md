# Skill: On-Page Optimization

## Purpose
Improve discoverability of existing content.

## Inputs
- Content artifact
- Target keyword cluster

## Process
1. Align headline and subheadings with search intent.
2. Optimize structural elements.
3. Improve clarity and relevance.

## Outputs
- Optimized content structure

## Constraints
- Must preserve original messaging.

## Invocation
Used when preparing content for search visibility.
