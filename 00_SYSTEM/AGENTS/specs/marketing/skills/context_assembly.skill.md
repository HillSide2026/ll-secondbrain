# Skill: Context Assembly

## Purpose
Assemble the required context for skill execution.

## Inputs
- Task definition
- Skill inputs
- Relevant artifacts
- Campaign or system context

## Process
1. Identify required inputs for the skill.
2. Retrieve relevant artifacts.
3. Assemble execution context.
4. Validate completeness.

## Outputs
- Structured execution context
- Skill-ready input package

## Constraints
- Context must come from authorized sources.
- Missing inputs must be flagged.

## Invocation
Used before skill execution.
