# Skill: Landing Flow Optimization

## Purpose
Improve landing page conversion effectiveness.

## Inputs
- Landing page structure
- Offer definition

## Process
1. Evaluate narrative flow.
2. Identify friction points.
3. Improve section sequencing.

## Outputs
- Optimized landing page structure

## Constraints
- Must maintain offer clarity.

## Invocation
Used during funnel optimization.
