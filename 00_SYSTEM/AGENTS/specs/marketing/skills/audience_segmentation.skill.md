# Audience Segmentation Skill

## Purpose
Define target audience segments and tie each segment to messaging requirements.

## Capabilities
- Segment audiences by role, industry, or problem
- Map audience to message
- Identify audience pain points
- Define audience-specific content needs

## Inputs
- Campaign goals
- Available customer and market context

## Outputs
- Audience segmentation matrix
- Segment-to-message mapping

## Boundary Rules
- Use approved positioning only.
- Do not infer policy exceptions from data gaps.
- Escalate if segmentation requires unsupported assumptions.
