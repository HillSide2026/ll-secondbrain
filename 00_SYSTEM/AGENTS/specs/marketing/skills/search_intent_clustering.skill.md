# Skill: Search Intent Clustering

## Purpose
Organize keywords and queries into clusters representing distinct user intents.

## Inputs
- Keyword list
- Search query patterns

## Process
1. Analyze search query semantics.
2. Group queries by user intent.
3. Identify dominant informational needs.
4. Map clusters to content opportunities.

## Outputs
- Intent clusters
- Content opportunity map

## Constraints
- Clusters must reflect real user intent patterns.

## Invocation
Used to identify search-driven content opportunities.
