# Channel Strategy Skill

## Purpose
Select and prioritize distribution channels for campaign execution.

## Capabilities
- Evaluate channel suitability
- Allocate channel priority
- Define channel-specific asset requirements
- Define cross-channel coordination

## Inputs
- Audience segments
- Campaign objective and funnel requirements

## Outputs
- Channel plan with priority tiers
- Channel dependency and coordination notes

## Boundary Rules
- External I/O must occur through approved integration adapters.
- Do not execute live publishing actions.
- Escalate when channel use conflicts with policy or capability limits.
