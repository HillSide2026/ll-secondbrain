# Skill: Marketing Repository Indexing

## Purpose
Maintain an index of marketing assets and campaign artifacts.

## Inputs
- Artifact metadata
- Campaign mapping

## Process
1. Register artifact in repository index.
2. Link artifact to campaign and template references.
3. Update searchable index.

## Outputs
- Updated repository index

## Constraints
- Index must reflect authoritative artifact states.

## Invocation
Used when assets are created, updated, or archived.
