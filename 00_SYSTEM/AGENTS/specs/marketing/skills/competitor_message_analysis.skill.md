# Skill: Competitor Message Analysis

## Purpose
Analyze messaging used by comparable firms or services.

## Inputs
- Competitor messaging
- Industry context

## Process
1. Extract core messaging themes.
2. Identify value propositions.
3. Identify positioning patterns.

## Outputs
- Competitor messaging report

## Constraints
- Analysis must remain objective.

## Invocation
Used when assessing market messaging landscape.
