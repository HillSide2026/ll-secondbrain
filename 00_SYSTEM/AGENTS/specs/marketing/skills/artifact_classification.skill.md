# Skill: Artifact Classification

## Purpose
Classify marketing artifacts within the repository.

## Inputs
- Marketing artifact
- Campaign context

## Process
1. Identify artifact type.
2. Assign category and metadata.
3. Link artifact to campaign.

## Outputs
- Classified artifact record
- Updated repository index

## Constraints
- No content modification.

## Invocation
Used by Repository Governance Agent during asset ingestion.
