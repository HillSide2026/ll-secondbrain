# Funnel Architecture Skill

## Purpose
Design a campaign funnel with explicit stages, conversions, and nurture flow.

## Capabilities
- Define funnel stages
- Define conversion objectives
- Map assets to funnel stages
- Identify lead capture mechanisms
- Define nurturing paths

## Inputs
- Campaign objective and audience segments
- Channel constraints

## Outputs
- Funnel stage map
- Conversion and nurture path specification

## Boundary Rules
- Do not bypass approved campaign strategy.
- Do not publish or distribute content.
- Escalate if required funnel decisions exceed assigned authority.
