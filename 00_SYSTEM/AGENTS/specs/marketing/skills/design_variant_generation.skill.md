# Skill: Design Variant Generation

## Purpose
Generate bounded design variants from a common approved brief.

## Inputs
- Base design draft
- Variant specification
- Channel constraints

## Process
1. Enumerate allowed variant dimensions (layout, crop, length, format).
2. Generate variants within approved constraints.
3. Preserve message integrity across variants.
4. Assign deterministic variant identifiers.

## Outputs
- Variant set
- Variant matrix and rationale

## Constraints
- No new claims or offers may be introduced.
- Variant count must stay within request limits.
- Variants must inherit core compliance requirements.

## Invocation
Used when campaign requires multiple asset variants.
