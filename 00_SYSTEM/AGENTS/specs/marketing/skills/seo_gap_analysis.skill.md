# Skill: SEO Gap Analysis

## Purpose
Identify content areas where discoverability opportunities exist.

## Inputs
- Existing content inventory
- Keyword clusters
- Competitor coverage

## Process
1. Compare keyword clusters to existing content.
2. Identify missing topics.
3. Identify underdeveloped coverage.

## Outputs
- SEO gap report
- Recommended content topics

## Constraints
- Must focus on high-relevance topics.

## Invocation
Used when evaluating discoverability strategy.
