# Skill: Social Thread Design

## Purpose
Structure long-form ideas into multi-post social narratives.

## Inputs
- Source content
- Platform format

## Process
1. Identify core message.
2. Break narrative into sequential posts.
3. Ensure logical progression.
4. Add engagement prompts.

## Outputs
- Structured social thread

## Constraints
- Message integrity must remain intact.

## Invocation
Used when converting long-form content to social narratives.
