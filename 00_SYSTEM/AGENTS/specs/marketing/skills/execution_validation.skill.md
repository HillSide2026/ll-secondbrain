# Skill: Execution Validation

## Purpose
Validate outputs produced by skill execution.

## Inputs
- Skill output
- Expected output structure
- Task objective

## Process
1. Evaluate completeness of output.
2. Verify output format.
3. Check alignment with task objective.
4. Flag inconsistencies or missing components.

## Outputs
- Validation result
- Execution quality assessment

## Constraints
- Validation cannot alter the output.
- Errors must be flagged for correction.

## Invocation
Used immediately after skill execution.
