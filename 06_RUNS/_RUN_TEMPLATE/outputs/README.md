# Outputs

Place run outputs here. Typical structure:

- outputs/brief.md
- outputs/gmail/<thread_id>_summary.md
- outputs/gmail/<thread_id>_commitments.md
- outputs/gmail/<thread_id>_draft.md
- outputs/sharepoint/<doc_id>_diff.md
- outputs/sharepoint/<new_doc_id>_context.md
- outputs/prep_packets/<event_id>.md
- outputs/prep_packets/<event_id>_followups.md

Optional subfolders (nice-to-have, not required)

Create these empty folders in the template if you want consistent paths:

06_RUNS/_RUN_TEMPLATE/outputs/gmail/.gitkeep
06_RUNS/_RUN_TEMPLATE/outputs/sharepoint/.gitkeep
06_RUNS/_RUN_TEMPLATE/outputs/prep_packets/.gitkeep
