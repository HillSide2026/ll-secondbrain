# Corporate Failure Modes

Documented patterns of what goes wrong in corporate matters, with mitigations.

---

## Purpose

Capture institutional knowledge about:
- Common mistakes
- Near-misses
- Client-caused delays
- Regulatory pitfalls

---

## Failure Mode Format

```markdown
# [FAILURE MODE NAME]

## Pattern
Brief description of what goes wrong

## Trigger Conditions
- Condition 1
- Condition 2

## Consequences
- Consequence 1
- Consequence 2

## Detection Signals
- Early warning 1
- Early warning 2

## Mitigation
- Prevention step 1
- Prevention step 2

## Recovery
- If it happens, do X
- Escalation path
```
