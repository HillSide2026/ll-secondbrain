# Corporate Issue Maps

Taxonomies of legal issues organized by subject matter.

---

## Purpose

Provide structured issue checklists for:
- Matter scoping
- Risk identification
- Completeness verification

---

```markdown
# [ISSUE MAP NAME]

## Scope
What area this covers

## Issue Taxonomy

### Category 1
- [ ] Issue 1.1

  - Sub-issue 1.1.2
- [ ] Issue 1.2

### Category 2
- [ ] Issue 2.1
```
