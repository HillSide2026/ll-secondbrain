# Solution Scope: CARF Controls and Testing

Component 9 of CARF_PROGRAM. Runs across the full program.

## Purpose

Establishes basic testing controls. Formal internal audit is not required.

## Controls

- Annual program review
- Sample testing:
  - Select 10–20 accounts
  - Re-perform classification manually

## Documentation

- Scope of testing
- Findings
- Remediation actions

## Included

- Annual review protocol
- Sample testing methodology
- Findings and remediation log

## Excluded

- Formal internal audit
- External audit engagement
