# Solution Scope: CARF Recordkeeping

Component 7 of CARF_PROGRAM. High-risk area if not properly implemented.

## Purpose

Establishes the records retention framework required to support CARF reporting
and CRA review.

## Records to Retain

- Self-certifications
- Changes to user data
- Transaction records
- Classification logic

## Control Principle

Reports must be reproducible from retained records.

## Included

- Recordkeeping policy and controls
- Retention schedule
- Reproducibility standard

## Excluded

- Reporting output generation (CARF_REPORTING_OUTPUT)
