# RPAA_THREE_YEAR_REVIEW

RPAA three-year review cycle for registered entities.

## Interfaces

- Invoked by: Scheduled review cycle, client request
- Produces: Review findings, renewal or update deliverables
- Consumes: Existing registration data, operational changes since last review
- Overlays available: AML_KYC_PROGRAM, RAILS, CRYPTO

## Operating Posture

Containment over completeness; explicit exit states; no assumed ongoing representation.
