# RPAA_REGISTRATION

RPAA registration process for clients requiring retail payment activities act registration.

## Interfaces

- Invoked by: Client engagement, regulatory requirement identification
- Produces: Registration deliverables
- Consumes: Client entity and activity information
- Overlays available: AML_KYC_PROGRAM, RAILS, CRYPTO

## Operating Posture

Containment over completeness; explicit exit states; no assumed ongoing representation.
