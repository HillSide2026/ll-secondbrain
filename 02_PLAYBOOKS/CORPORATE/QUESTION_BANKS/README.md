# Corporate Question Banks

Structured questions for intake, diligence, and risk assessment.

---

## Categories

- **Intake** — Initial client meeting questions
- **Diligence** — Document and information requests
- **Risk** — Red flag identification

---

## Question Bank Format

```markdown
# [QUESTION BANK NAME]

## Context
When to use this bank

## Questions

### Section 1
1. Question text?
   - Follow-up if yes: ...
   - Follow-up if no: ...
   - Red flag: ...

### Section 2
1. Question text?
```
