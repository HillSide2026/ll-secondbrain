### Profitability Lens

For each real matter, classify the work into one of five postures:

| Posture | Meaning | Firm response |
|---|---|---|
| Grow | Profitable, repeatable, strategically aligned demand | Protect capacity, improve throughput, ask for referrals, package the offer |
| Price up | Valuable but underpriced or too bespoke | Reprice, narrow scope, require deposits, define change orders |
| Delegate | Work is valuable but does not require ML1 for every step | Push to clerk/freelancer/templates with ML1 review gates |
| Collect / pause | Work may be valid but cash discipline is weak | Collect AR, refresh retainer, pause new work until payment is clean |
| Exit / close | Low-value, stale, noisy, or non-strategic work | Close file, decline expansion, or convert to paid diagnostic only |

### Non-ML1 Capacity Lens

| Capacity Need | Use when | Candidate owner |
|---|---|---|
| Intake / qualification | Client demand is unclear, budget is unknown, or scope is not priced | Assistant / intake support / system checklist |
| Matter admin | Work is blocked by signatures, documents, reminders, retainers, AR, or file closing | Admin support |
| First-pass legal production | Work is repeatable, precedent-driven, or checklist-driven before ML1 review | Clerk / lawyer contractor / freelancer |
| Marketing execution | Work involves publishing, CRM hygiene, follow-up, or funnel maintenance | Marketing contractor / VA / system |
| Finance / collections | Work should not expand until payment, retainer, or AR status is clean | Bookkeeper / admin / finance support |
