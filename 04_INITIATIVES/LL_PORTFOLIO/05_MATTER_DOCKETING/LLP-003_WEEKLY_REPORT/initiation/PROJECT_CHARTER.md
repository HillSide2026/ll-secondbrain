# Project Charter

Project ID: LLP-003
Project Path: 05_MATTER_DOCKETING/LLP-003_WEEKLY_REPORT
Project Type: Operational Project
Stage: Initiating

## 1. Purpose
- Govern the weekly matter-docketing report so ML1 receives a reliable delivery-facing operating summary without changing systems of record.

## 2. Nature of Project
- Operational reporting/control packet focused on a recurring weekly view of matter state, activity periods, blockers, and next actions.

## 3. Strategic Rationale
- Weekly reporting is a core visibility surface for delivery operations; it should be governed as a packet with explicit boundaries rather than treated as an implied byproduct.

## 4. High-Level Deliverable
- A defined weekly-report packet covering report scope, inputs, cadence, review ownership, and escalation boundaries.

## 5. Authority
Final approval authority: ML1.

## 6. Promotion Path
- Advance to Planning once ML1 confirms the report purpose, inputs, cadence, and the minimum schema required for governed weekly output.
