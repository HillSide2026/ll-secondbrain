# Communication Plan

Project ID: LLP-025
Project Path: 08_MARKETING/LLP-025_MARKETING_STRATEGY
Stage: Planning

## Review Cadence

| Review | Frequency | Owner | Scope |
|---|---|---|---|
| Strategy state review | Monthly | ML1 | Positioning integrity, funnel portfolio status, KPI performance vs. targets, agent output quality |
| KPI review | Monthly | ML1 | Funnel-level KPIs for F01, F02, F03; CPL, show rate, diagnostic/entry-offer conversion, retainer conversion |
| Agent output review | Per output batch | Strategic Editor → ML1 | All published content and conversion changes reviewed by Strategic Editor before ML1 sign-off |
| F01 performance review | Monthly | ML1 | CPC, CPL trends; flag if CPL exceeds provisional threshold |
| F02 milestone review | At each milestone | ML1 | Offer definition, build progress, launch readiness |
| Planning gate decision | One-time | ML1 | Planning → Executing authorization after all planning artifacts are complete |

## Escalation Protocol

| Trigger | Escalation Path | Response Time |
|---|---|---|
| F01 CPL exceeds provisional threshold | Agent flags → ML1 review | Within 5 business days |
| Agent output contradicts positioning | Strategic Editor blocks; flags to ML1 | Before any publication |
| F02 build stalls beyond milestone date | ML1 notified; milestone date revised | Within 5 business days of missed milestone |
| ICP drift detected in intake profile | LLP-014 flags to LLP-025 owner (ML1) | Next scheduled strategy review |
| New funnel proposal generated | ML1 review and approval required before any work proceeds | Before any work begins |
| F01 wind-down consideration triggered | ML1 decision required; F02 performance data presented | Dedicated decision session |

## Communication Boundaries

- Agents do not communicate positioning changes, ICP changes, or KPI target changes externally. These require ML1 approval.
- No funnel launch, major offer change, or agent activation is communicated to external channels without ML1 approval.
- Strategic Editor is the internal coherence gate; ML1 is the external authorization gate.
