# Assumptions & Constraints

Project ID: LLP-025
Project Path: 08_MARKETING/LLP-025_MARKETING_STRATEGY
Stage: Planning

## Assumptions

| ID | Assumption | Impact if Wrong |
|---|---|---|
| A-01 | 30+ days of baseline KPI data is usable for provisional target-setting | Phase 2 and Phase 3 KPI formalization must be delayed until clean baseline is available |
| A-02 | The Corporate Health Check concept is sufficiently defined to move to offer-scoping; not starting from zero | F02 build workstream scope expands materially; launch milestone pushed |
| A-03 | F01 (Google Ads) remains active and generating some qualified intake during the planning stage | Revenue risk if F01 is disrupted; planning stage proceeds without viable bridge channel |
| A-04 | ML1 has capacity to review and approve planning artifacts and KPI targets within the planning window | Planning gate is delayed; execution authorization cannot proceed |
| A-05 | F03 (Payments/MSB) continues operating during planning; payments vertical authority build is not paused | Payments vertical falls behind; F03 conversion data is unavailable for KPI formalization |
| A-06 | Website / landing page infrastructure is available or buildable to support F02 launch | F02 launch milestone requires platform work not currently scoped |
| A-07 | ICP-02 (mature Ontario operating companies) can be reached through at least one scalable channel once identified | If no channel exists for ICP-02, the firm's differentiated positioning cannot be operationalized through marketing |

## Hard Constraints

| ID | Constraint | Source |
|---|---|---|
| C-01 | No new funnel may be created without: defined ICP, defined entry offer, defined measurable KPIs, and upstream objective mapping | Marketing governance rules (`MARKETING_STRATEGY.md`) |
| C-02 | Marketing does not provide legal advice or accept matters autonomously | Compliance and professional conduct |
| C-03 | All ICP changes require ML1 approval | ML1 authority statement |
| C-04 | All KPI target changes require ML1 approval | ML1 authority statement |
| C-05 | Positioning overrides metrics — no metric optimization that dilutes ICP quality or brand posture is authorized | Agent authority hierarchy |
| C-06 | F01 wind-down may not proceed without explicit ML1 approval and defined F02 performance baseline | Capacity risk management |
| C-07 | No agent may change positioning, ICP definition, or KPI targets autonomously | Agent scope boundaries |
