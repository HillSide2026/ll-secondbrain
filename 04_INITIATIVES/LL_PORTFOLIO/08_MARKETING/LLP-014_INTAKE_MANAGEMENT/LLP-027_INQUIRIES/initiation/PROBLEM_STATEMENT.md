# Problem Statement

Project ID: LLP-027
Project Path: 04_INITIATIVES/LL_PORTFOLIO/08_MARKETING/LLP-014_INTAKE_MANAGEMENT/LLP-027_INQUIRIES

Incoming inquiries arrive through multiple channels — web forms, email, and
referral — with no governed triage or response system in place. There is no
defined SLA for initial response, no routing logic for practice area
classification, and no formal handoff protocol to the consultation stage.

The consequences are concrete: inquiries that are not acknowledged promptly
go cold; inquiries that are not routed correctly get delayed or mishandled;
inquiries that qualify for consultation have no defined pathway to get there.
The firm has no pipeline visibility at the earliest intake stage because there
is nothing logging or classifying what arrives.

This is the first stage of the intake pipeline. If it is not governed, the
pipeline cannot produce reliable inquiry-to-consult conversion regardless of
how well the upstream funnels and lead capture operate.

This project initiates the work to govern inquiry receipt through to handoff.
