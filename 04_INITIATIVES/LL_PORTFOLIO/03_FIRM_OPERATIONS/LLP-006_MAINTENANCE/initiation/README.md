# LLP-006 — Matter Maintenance

## Scope

LLP-006 governs the ongoing maintenance of communication, documentation, and information
associated with open matters. It defines and enforces the protocols by which matter-related
data — across Gmail, Google Calendar, Clio, and SharePoint — is classified, organized,
filed, and made accessible to agents and staff.

**In scope:**

| System | Maintenance function |
|--------|---------------------|
| **Gmail** | Inbox triage; state and matter label classification; thread-to-matter linking |
| **Google Calendar** | Event classification; matter linkage for calendar events; scheduling records |
| **Clio** | Matter record maintenance; synchronization of matter status between Clio and secondbrain |
| **SharePoint (LegalMatters)** | Document provenance classification; filing structure compliance; matter document inventory |

**Out of scope:**

- Cross-system matter linking (Gmail ↔ Clio ↔ SharePoint) — belongs to LLP-005
- New matter intake — separate project
- Client-facing communications — governed by output labeling policy (PRO-001/POL-001)
- System infrastructure (credentials, API access) — system-level concern
