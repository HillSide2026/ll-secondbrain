# Project Plan

Project ID: `LLP-033`
Project Path: `04_INITIATIVES/LL_PORTFOLIO/07_GROWTH_PROJECTS/LLP-033_ASSOCIATE_LAWYER`
Stage: `Planning`

## Planning Objective

Turn the associate-lawyer concept into an execution-ready ML1 decision packet,
not a vague staffing idea.

## Current Planning Focus

1. define the actual role and candidate profile
2. define exactly what work can and cannot be delegated
3. define how supervision and review will work in practice
4. define monitoring and analytics needed to detect failure early
5. define the economics gate for delegated delivery

## Immediate Planning Sprint

- define the candidate profile for the first delivery-lawyer role
- define the first matter classes that are eligible for delegated delivery
- define the minimum supervision and sign-off sequence
- define the monitoring dashboard and exception log
- define the delegated `F01` contribution formula
- define what would count as a no-go outcome

## Completion Condition

Planning is complete when ML1 can clearly answer:
- who the role is for
- what work is delegable
- how it will be supervised
- how it will be monitored
- whether the economics justify activation
