# Metrics

Project ID: `HBP-011`
Stage: `Planning`

## Decision-Readiness Metrics

| Metric | Definition | Target | Evidence |
| --- | --- | --- | --- |
| Boundary clarity | Separation between agency consulting and Levine Law legal work is explicit | boundary rules documented clearly enough for ML1 review | `ASSUMPTIONS_CONSTRAINTS.md`, `DECISION_FRAME.md` |
| Option set clarity | Number of credible operating paths still under active consideration | narrowed to one preferred option, with at most one fallback | `DECISION_FRAME.md` |
| Channel credibility | At least one believable buyer/channel path is documented | one credible path documented clearly | `DEPENDENCIES.md`, `DECISION_FRAME.md` |
| Economics threshold | Minimum viable economics are defined and compared to the preferred path | threshold defined and preferred option assessed against it | `METRICS.md`, `RISK_REGISTER.md` |
| Operating-model clarity | Minimum delivery model and capability assumptions are explicit | minimum model documented clearly enough for ML1 review | `ASSUMPTIONS_CONSTRAINTS.md`, `DEPENDENCIES.md` |
| ML1 gate readiness | Planning packet is complete enough for ML1 decision | all required planning artifacts drafted | planning folder and `../APPROVAL_RECORD.md` |

## Measurement Rule

These metrics are used to determine whether planning has produced a real
decision or only more conceptual noise.
