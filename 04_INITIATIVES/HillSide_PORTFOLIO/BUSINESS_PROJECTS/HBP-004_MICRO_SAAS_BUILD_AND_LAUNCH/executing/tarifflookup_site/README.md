# TariffLookup Site Bundle

This folder contains a static landing page bundle for `tarifflookup.ca`.

## Upload to HostGator

1. Open HostGator `cPanel`.
2. Open `Domains` and confirm the document root for `tarifflookup.ca`.
3. Open `File Manager`.
4. Enter the document-root folder for `tarifflookup.ca`.
5. Upload the contents of this folder into that document root.

Upload these files:
- `index.html`
- `styles.css`
- `privacy.html`
- `robots.txt`
- `sitemap.xml`
- `404.html`

## Notes

- `index.html` must remain in the document root so it loads as the home page.
- The current CTA uses `mailto:hello@tarifflookup.ca`.
- If you prefer a different address, replace `hello@tarifflookup.ca` in `index.html` and `privacy.html` before uploading.
