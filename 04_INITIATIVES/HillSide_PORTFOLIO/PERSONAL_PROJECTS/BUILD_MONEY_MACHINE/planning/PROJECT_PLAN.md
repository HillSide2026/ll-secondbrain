## Phase Structure

### Phase 1 - Baseline and Control Setup

- define the ideal-day and lifestyle destination the machine is serving
- capture current monthly income and expense reality
- define basic-expense versus desired-lifestyle expense targets
- define expense categories and control rules
- establish the first monthly dashboard version

### Phase 2 - Wealth System Design

- define the emergency-fund rule
- define the savings-rate target and contribution order
- define the long-term investing allocation logic
- define the income-engine map and financial-independence number
- define stage-qualification criteria for stability, security, and independence

### Phase 3 - Long-Horizon Path Model

- define progression criteria for freedom and abundance
- define the gap from current passive / portfolio income to desired coverage
- translate the 2030 and 2040 net-worth targets into a tracked path
- define review thresholds and adjustment triggers
- prepare the stage-gate packet for executing
