# Executing Workplan

Executing is not yet authorized. This file defines the intended execution model
so the stage-gate packet is concrete before the stage opens.

## Execution Phases

### Phase 1 - Vision and Truth

| Sequence | Exercise | Purpose | Evidence |
| --- | --- | --- | --- |
| 1 | Vision Creation Exercise | Define the destination the machine should serve | Written vision and ideal-day description |
| 2 | Financial Truth Exercise | Establish current financial reality | Income, expenses, debts, and net-worth baseline |
| 3 | Money Flow Mapping | Map every dollar and find leaks | Spending-category map and leak list |
| 4 | Expense Alignment Exercise | Compare spending to the vision and values | Keep / cut / redirect decisions |

### Phase 2 - Engine Design

| Sequence | Exercise | Purpose | Evidence |
| --- | --- | --- | --- |
| 5 | Income Engine Identification | Define how money currently and eventually enters the system | Active / portfolio / passive income map |
| 6 | Wealth Priority Allocation | Assign percentages and order of use to income | Living / saving / investing / giving allocation rules |
| 7 | Investment Strategy Definition | Create the long-term investment process | Asset-type plan, cadence, and automation rule |
| 8 | Financial Independence Number | Define the "enough for necessities" threshold | Basic-expense target and independence number |
| 9 | Gap Analysis Exercise | Measure current asset-income position against future requirements | Gap-to-independence and gap-to-freedom model |

### Phase 3 - Operating Cadence

| Sequence | Exercise | Purpose | Evidence |
| --- | --- | --- | --- |
| 10 | Monthly Money Machine Review | Keep the system running and make adjustments based on evidence | Monthly review log and action list |

## First 30-Day Launch Pattern

- Week 1: complete the vision creation and financial truth exercises
- Week 2: complete money flow mapping and expense alignment
- Week 3: lock the income engine map, priority allocation, and investment strategy
- Week 4: calculate the independence number, run gap analysis, and complete the first monthly review

## Ongoing 90-Day Pattern

- Month 1: establish clarity and first control changes
- Month 2: improve surplus, automation, and reserve discipline
- Month 3: validate stage qualification, refine the gap model, and decide what changes next

## Continue / Adjust / Stop Rule

- continue when surplus, review discipline, and stage logic are improving
- adjust when the system is running but friction or misalignment is visible
- stop and redesign when the process becomes too complex to sustain or the
  stated vision has materially changed
