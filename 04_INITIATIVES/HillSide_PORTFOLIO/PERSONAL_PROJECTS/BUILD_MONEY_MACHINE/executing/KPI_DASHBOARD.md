# KPI Dashboard

Executing is not yet authorized. This file defines the dashboard fields that
should be live once the stage opens.

## Control-Layer Dashboard

| Metric | Purpose | Cadence |
| --- | --- | --- |
| Total income | Track inflow strength | Monthly |
| Total expenses | Track outflow reality | Monthly |
| Monthly surplus | Confirm there is money left to direct | Monthly |
| Savings rate | Measure wealth-building discipline | Monthly |
| Investment contribution amount | Measure execution of the investing system | Monthly |
| Emergency-fund coverage | Measure resilience | Monthly / Quarterly |
| Net worth | Measure long-horizon accumulation | Monthly / Annual |

## Direction-Layer Dashboard

| Metric | Purpose | Cadence |
| --- | --- | --- |
| Basic living expense target | Anchor the independence threshold | Quarterly / Annual |
| Desired lifestyle expense target | Anchor the freedom threshold | Quarterly / Annual |
| Passive / portfolio income | Measure asset-driven cash flow | Monthly / Quarterly |
| Coverage of basics | Measure progress toward independence | Monthly / Quarterly |
| Coverage of desired lifestyle | Measure progress toward freedom | Quarterly / Annual |
| Highest stage currently qualified | Prevent vague self-assessment | Monthly / Quarterly |
| Monthly review completion | Keep the machine operational | Monthly |

## Monthly Signal Prompts

- biggest spending leak identified this month
- most aligned spend this month
- next highest-leverage change
- current stage: stability, security, independence, freedom, or abundance
- continue / adjust / stop decision
