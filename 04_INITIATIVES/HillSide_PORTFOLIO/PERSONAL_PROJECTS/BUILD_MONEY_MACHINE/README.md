# Build Money Machine

This project is a planning-stage HillSide personal project.

## Project ID

`BUILD_MONEY_MACHINE`

## Current State

- Program branch: `PERSONAL_PROJECTS`
- Stage: `Planning`
- `Initiating -> Planning` gate approved on `2026-03-26`
- Planning packet is open in `planning/`
- Executing is not yet authorized
- This is a personal project and not an `HBP` business-project packet

## Objective

Design and implement a structured, repeatable system for generating, managing,
and growing personal wealth through disciplined cash flow, intentional
spending, long-term investing, and explicit lifestyle design so that ML1 can
move through five stages: financial stability, financial security, financial
independence, financial freedom, and financial abundance.

## Baseline Objectives

- net worth of at least CAD 2,000,000 by `2030-12-31`
- net worth of at least CAD 4,000,000 by `2040-12-31`

## Money Machine Stage Model

1. Financial Stability: bills are paid, survival debt is avoided, and a small
   cushion exists.
2. Financial Security: a real reserve exists and immediate threats are reduced.
3. Financial Independence: assets can cover basic living expenses.
4. Financial Freedom: asset income can support the desired lifestyle.
5. Financial Abundance: wealth exceeds lifestyle needs and can support impact,
   contribution, and legacy.

## Current Planning Workstreams

- define the planning-ready personal cash-flow and expense-management system
- define the savings, investing, and allocation model
- define the life-vision, lifestyle target, and "enough" numbers the machine is meant to serve
- define the monthly dashboard, metrics, and review cadence
- define the qualification rules for the five money-machine stages
- define the path from current baseline to the `2030-12-31` and `2040-12-31` targets
- prepare the `Planning -> Executing` gate packet for ML1

## Notes

- Use this project for personal wealth-building strategy, operating discipline,
  and review cadence.
- High-risk speculation, short-term trading, and get-rich-quick strategies are
  out of scope for this project.
