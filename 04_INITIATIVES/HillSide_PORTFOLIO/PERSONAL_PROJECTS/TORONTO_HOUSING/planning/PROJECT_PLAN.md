## Decision Use

Use this file to sequence planning work for the Toronto housing decision
project and to define what must be true before ML1 closes the planning stage.

## Planning Objective

Produce a decision-ready recommendation on whether to advance with new owned
housing in Toronto or to leave Toronto.

## Current Planning Focus

1. lock the real option set and decision criteria
2. formalize the housing budget inside `HBP-001`
3. compare the two real paths against fit, liquidity, and flexibility rules
4. assemble a clean `Planning -> Executing` packet for ML1
