# Quality Check (Operational)

Checklist (must pass before release or promotion):
- Output labeling present. Ref: 01_DOCTRINE/02_policies/POL-001_Output_Labeling_Requirement.md (POL-001)
- Provenance present. Ref: 01_DOCTRINE/02_policies/POL-002_Provenance_Requirement.md (POL-002)
- No novel policy content. Ref: 01_DOCTRINE/02_policies/POL-003_Novel_Policy_Prohibition.md (POL-003)
- External/client-facing output has ML1 approval. Ref: 01_DOCTRINE/02_policies/POL-004_External_Output_Approval_Requirement.md (POL-004)
- Low-confidence items escalated. Ref: 01_DOCTRINE/02_policies/POL-005_Low_Confidence_Escalation.md (POL-005)
- Promotion to Active requires indexing. Ref: 01_DOCTRINE/02_policies/POL-011_Discoverability_Requirement.md (POL-011)
- Version changes include supersession metadata where applicable. Ref: 01_DOCTRINE/02_policies/POL-010_Maintainability_Versioning_Requirement.md (POL-010)
- Conflicts at same layer resolved with explicit supersession. Ref: 01_DOCTRINE/02_policies/POL-007_Consistency_Supersession_Requirement.md (POL-007)
