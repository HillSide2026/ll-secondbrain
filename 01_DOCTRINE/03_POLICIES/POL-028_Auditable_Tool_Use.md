# POL-028 — Auditable Tool Use

Policy Statement: Each integration adapter call must be recorded in the run container with sufficient metadata to identify the tool, action, timing, and governing run context.
Authority (Principles referenced): PRN-026, PRN-029
Enforcement expectation: Adapter calls without required run metadata are non-compliant and must be flagged and remediated.
