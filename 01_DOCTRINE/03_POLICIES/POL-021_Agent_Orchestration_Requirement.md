# POL-021 — Agent Orchestration Requirement

Policy Statement: Workers may be invoked only by an authorized agent acting within a governed workflow.
Authority (Principles referenced): PRN-022, PRN-023, PRN-027
Enforcement expectation: Worker execution without agent mediation in a governed workflow is non-compliant and must be blocked.
