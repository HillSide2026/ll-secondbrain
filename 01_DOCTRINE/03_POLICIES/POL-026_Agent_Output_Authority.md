# POL-026 — Agent Output Authority

Policy Statement: Only the authorized issuance layer may classify artifacts as Authorized Outputs; agent issuance requires ML1 and LL sign-off before Authorized Output status is valid.
Authority (Principles referenced): PRN-020, PRN-025
Enforcement expectation: Authorized Output classification outside the authorized issuance layer, or without required ML1 and LL sign-off, is non-compliant and must be rejected or escalated.
