# POL-027 — Run Record Requirement

Policy Statement: Every governed execution must create or attach to a run container that records the responsible agent, invoked workers, integration calls, intermediate artifacts, and resulting outputs.
Authority (Principles referenced): PRN-026, PRN-029
Enforcement expectation: Governed executions lacking a run container record are non-compliant and must be remediated or invalidated.
