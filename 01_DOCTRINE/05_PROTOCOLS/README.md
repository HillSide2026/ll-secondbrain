# Protocols (Level 4)

Definition: Operational enforcement rules that implement policies as conditional logic.
Protocols are interaction contracts that define process choreography between components.
Boundary: Do not introduce new policy requirements or reinterpret principles.

Typical examples:
- Change Proposal Protocol
- Doctrine Update Protocol
- Template Release Protocol
- Exception Escalation Protocol
