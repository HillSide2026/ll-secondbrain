# LLPRO-001 — Intake Qualification Protocol

**Trigger:** Any new prospect, inquiry, or referral — before a matter is opened in Clio.

---

## Purpose

Defines the three sequential gates that must pass before a matter is opened. Each gate is a hard stop if it fails; subsequent gates are not assessed until the prior gate passes.

---

## Gates

### Gate 1 — Capacity Check (LLPRN-001)

Is ML1 capacity available to take on a new matter?

- If at ceiling: stop. Refer or defer. Do not proceed to Gate 2.
- If capacity is available: proceed to Gate 2.

### Gate 2 — ICP Fit (LLPRN-002, LLPOL-004)

Does the prospect match ICP-01 or ICP-02?

- **ICP-01:** Ontario operator, CAD $5M+ cash flow, accountant involved, not in acute crisis.
- **ICP-02:** Payments/MSB/fintech operator with Canadian regulatory exposure.
- If fit: proceed to Gate 3.
- If no fit: refer or decline unless ML1 approves an exception per LLPOL-004 (documented rationale required at the matter level before proceeding).

### Gate 3 — Retainer Sizing (LLPOL-003)

What retainer is appropriate for this matter's scope and complexity?

- Minimum CAD $5,000 for new clients.
- Complex or urgent matters carry a higher retainer calibrated to the matter.
- If the prospect will not meet the floor: decline.
- If sizing is agreed: proceed to engagement.

---

## Outputs

| Result | Condition |
|---|---|
| Proceed to engagement | All three gates pass |
| Refer or decline | Gate 1 or Gate 2 fails with no exception |
| ML1 exception review | Gate 2 borderline or below-floor retainer request |
