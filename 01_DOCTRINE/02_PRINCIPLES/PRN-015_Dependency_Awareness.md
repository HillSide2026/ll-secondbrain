# PRN-015 - Dependency Awareness

Title: Dependency Awareness

Statement:
Artifacts should acknowledge and respect their dependencies to prevent hidden conflicts across the system.

Example:
Protocols referencing policies should not implicitly reinterpret them.

Rationale:
Explicit dependency awareness prevents silent doctrinal drift and cross-layer contradiction.
