# PRN-028 - Capability Containment

Title: Capability Containment

Statement:
Each runtime component should act only within its assigned capability boundary, with ambiguity resolved by escalation rather than expansion.

Rationale:
Containment prevents scope creep and enforces disciplined escalation behavior.
