# PRN-029 - Traceable Mediation

Title: Traceable Mediation

Statement:
Tool use, worker activity, and output generation should remain attributable to the orchestrating agent and the governing doctrine under which the run occurred.

Rationale:
Traceable mediation preserves accountability across mediated execution chains.
