# PRN-027 - Layer Fidelity

Title: Layer Fidelity

Statement:
Agents, workers, integration adapters, and run containers should remain functionally distinct and should not collapse into one another's roles.

Rationale:
Role fidelity limits coupling and keeps system behavior legible under change.
