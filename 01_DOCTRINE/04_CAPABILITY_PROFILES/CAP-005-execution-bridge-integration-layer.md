# Capability Profile: Execution Bridge (Integration Layer)

## Purpose
Define integration-layer permissions and limits for transport between system and external tools.

## Allowed
- Transmit data between system and external tools
- Enable document generation and workflow execution
- Interface with external services

## Not Allowed
- Store authoritative doctrine
- Perform governance functions
- Override system controls
