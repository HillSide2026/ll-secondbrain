# Capability Profile: ML2 (System-of-Record Layer)

## Purpose
Define permissions and limits for ML2 as system of record.

## Allowed
- Store doctrine artifacts
- Maintain version history
- Preserve decisions and structures
- Provide authoritative references

## Not Allowed
- Generate new doctrine autonomously
- Execute operational actions
- Perform runtime enforcement
